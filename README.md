# Галерея изображений с AJAX-загрузкой

Простое веб-приложение на чистом PHP и JavaScript (без фреймворков), позволяющее загружать изображения через AJAX без перезагрузки страницы и просматривать их в виде галереи. В PHP используется объектно-ориентированный подход.

## Возможности

- **Загрузка изображений** – выбор файла через кнопку «Обзор», автоматическая отправка на сервер с отображением сообщения об успехе или ошибке.
- **Галерея** – все загруженные изображения выводятся в виде квадратной сетки.
- **Поддерживаемые форматы** – JPG, PNG, GIF, WEBP.
- **Ограничения** – максимальный размер файла 5 МБ.

## Требования

- PHP 7.4 или выше
- Расширение `fileinfo` (обычно включено по умолчанию)
- Веб-сервер (Apache, Nginx и т.д.)
