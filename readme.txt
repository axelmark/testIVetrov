Необходимо на чистом PHP и JS с использованием ООП в PHP создать две страницы.
Первая страница: Загрузка картинки по ajax без перезагрузки страницы.
На странице должна отображаться кнопка «Обзор» куда можно добавить картинку, после того как пользователь выбирает изображение оно автоматически загружается на сервер
и пишется сообщение о том что картинка успешно загружена.

Вторая страница: Вывод всех загруженных картинок.
Выводим в виде квадратной сетки все загруженные ранее картинки.
